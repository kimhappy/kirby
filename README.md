# Kirby
Library for black-box modeling of complex non-linear audio system.

## Prerequisite
- [UV](https://github.com/astral-sh/uv)

## Build
```sh
uv build
```
This will generate `.whl` and `.tar.gz` files in the `dist/` directory.

## Use CLI
See [the CLI document](kirby-cli/README.md).
